# CVSS Severity Thresholds

## Recommended Filters
- Critical (9.0-10.0): Always investigate immediately
- High (7.0-8.9): Investigate within first test day
- Medium (4.0-6.9): Check if affected component is in use
- Low (0.1-3.9): Background context only

## Environment Variable Mapping
INTEL_CVSS_MIN=7.0 → Only High and Critical
INTEL_CVSS_MIN=4.0 → Medium, High, Critical
INTEL_CVSS_MIN=0.0 → All severities (default)

## Bug Bounty Relevance
Most bounty programs pay for:
- Remote Code Execution (RCE): CVSS 9.0-10.0
- SQL Injection / SSRF: CVSS 7.0-9.0
- XSS / IDOR: CVSS 4.0-7.0
- Info Disclosure: CVSS 0.0-4.0 (sometimes informational only)
